# mendeleev-ebal

Таблица Менделеева ебущая тебя в рот

## how to instal
============================================================

Сначала нужно сделать git clone
> git clone https://github.com/hackwork-tymm0/mendeleev-ebal.git

Переходим в папку с проектом
> cd %PATH_TO_PROJECT%

И запускаем
> node main.js
